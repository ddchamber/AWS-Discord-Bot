This is a Discord Bot to answer questions about how to get started with dropshippping through the Discord channel space. 
It is currently connected to AWS, so there is no activation required on the user's part.
Simply type in a question related to dropshipping into the Chatbot discord channel, and the bot will respond within a few seconds. 
The bot currently uses Claude as a LLM and SQLite as a database to hold the RAG files used to answer questions.
It is grounded using discord messages and scripts from Mike Chamberlin and Dom Dei's dropshipping course and discord question service.
Please reach out with any other questions!
